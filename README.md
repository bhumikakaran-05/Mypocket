💸 MyPocket: Smart Student Expense Tracker

**MyPocket** is a modern, mobile-first web application designed specifically for students and individuals managing a fixed monthly allowance. It moves away from complex accounting and focuses on what matters most: **"How much pocket money do I have left?"**

Built with a clean, teal-themed aesthetic, MyPocket provides a "crafted" user experience with smooth animations and intuitive data visualization.

__🚀 Key Features:__

1. Monthly Pocket Money Focus: Set your total allowance at the start of the month and watch your balance update in real-time as you spend.
2. Smart Savings Logic: Automatically calculates your monthly savings. If you overspend, the app highlights exactly how much you've gone over your limit.
3. Dynamic Categories: Comes with student-centric defaults (Rent, WiFi, Food, Gas, etc.). Add, edit, or remove categories every month to match your changing lifestyle (e.g., adding a "Gym" category or removing "Travel").
4. Yearly Trends & Analytics: A dedicated Year Summary dashboard with interactive bar charts to visualize your spending habits and identify overspent months.
Persistent Storage: Uses localStorage to keep your data safe on your device without needing a complex backend or account.
5. Privacy First: All financial data stays strictly on your local machine.
   
__🛠️ Tech Stack:__

Frontend: React 18 + TypeScript
Styling: Tailwind CSS (Utility-first, modern UI)
Animations: Framer Motion (Smooth transitions and modal effects)
Charts: Recharts (Professional data visualization)
Icons: Lucide React
Date Handling: date-fns

__📸 UI/UX Highlights:__

Glassmorphism Elements: Modern card designs with soft shadows and subtle gradients.
Responsive Design: Optimized for mobile use but scales beautifully to desktop.
Interactive Feedback: Progress bars change color (Green → Orange → Red) based on your budget usage to provide instant visual warnings.

__🏁 Getting Started:__

Clone the repo: https://github.com/bhumikakaran-05/Mypocket.git
Install dependencies: npm install
Run the app: npm run dev
Build for production: npm run build
